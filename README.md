# 软件介绍
提供桌面提醒功能：H5通知和声音提醒。

## 添加提醒
可通过两种方式添加提醒事项
1. `+XXX 通知名称`
2. `2019-05-22 10:00:00 通知名称`

第一种方式表示XXX秒后提醒
第二种方式表示将在指定时刻提醒
**两种方式可能都会有一点误差**

## 状态更新
每个提醒有4个状态
+ 待执行
  + 未到时间的任务
+ 已执行
  + 已过执行时间且通知正常执行
+ 已过期
  + 可能是在提醒时间utool未运行或者其他原因
+ 已删除
  + 点击提醒后面的删除，删除功能将会从utools.db中删掉

## 其他
快速输入当前日期：在子输入框中的最前面输入`@TODAY`（不区分大小写）

快速输入当前时间，不包含秒：在子输入框中的最前面输入`@NOW`（不区分大小写）
